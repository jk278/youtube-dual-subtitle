# Youtube dual subtitle / Youtube 双语字幕全平台

Youtube 双语字幕。移动端（mobile）修复，双端适用，而且支持 Via 浏览器。

## 不知道你是否和我陷入相同的困境：
1. 想为 Youtube 开启双语字幕。
2. 喜欢 Firefox 和 Via 浏览器，不想使用不太稳定的 Firefox Nightly。
3. Firefox for Android 和 Via 支持的油猴脚本中，大部分双语字幕在移动端并不适用。

### 如果你的回答是：
> Yes!

那么恭喜你，这个脚本为你量身定制！
